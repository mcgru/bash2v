# Documentation

This directory contains technical documentation for the Bash-to-Vlang transpiler.

## Documents

| File | Description |
|------|-------------|
| `bash-grammar.md` | Bash v5.1 grammar reference |
| `vlang-mapping.md` | Bash to Vlang construct mapping |
| `coverage.md` | Feature coverage matrix |
| `constraints.md` | Implementation constraints and requirements |

## For Contributors

1. Read `bash-grammar.md` to understand supported Bash features
2. Check `vlang-mapping.md` for translation patterns
3. Review `coverage.md` to see what's already implemented
4. Add new mappings following existing conventions

## For Users

- See `../README.md` for quick start guide
- Check `../examples/` for sample conversions
- Review `../tests/` for test cases
